Decision Matrix example

Create outcomes from multiple lists of options, including wildcards (*). See the tables.

Click random, or select options, then submit.

Code
options - used to build the select values 

                    let options = {
                        adjective: ['sad', 'furtive', 'cautious', 'happy', 'adventurous'],
                        color: ['red', 'green', 'blue'],
                        animal: ['dog', 'cat', 'rabbit', 'rooster', 'gazelle', 'aardvark', 'sea lion']
                    };
outcomes - acceptable values for each option [0,1,2] and the text that is returned [3] 

                    let outcomes = [
                        [
                            ['sad', 'cautious', 'furtive'],
                            ['blue'],
                            ['*'],
                            ['Can I give you a hug?']
                        ],
                        [
                            ['cautious'],
                            ['*'],
                            ['cat'],
                            ['Is meow a good time to talk?']
                        ],
                        ...
                    ];
